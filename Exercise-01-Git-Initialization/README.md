## Objective
To understand basic Git operations such as initializing a repository, committing files, connecting to GitHub, and pushing changes.

## Steps Performed

2. Created a README file and added content.

4. Created a GitHub repository.
5. Connected local repository to remote GitHub repository.
6. Pushed commits to GitHub.
7. Verified remote connection by pulling updates.
