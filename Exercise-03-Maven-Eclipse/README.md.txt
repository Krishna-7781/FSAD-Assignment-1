# Exercise 3: Creating and Running a Maven Project in Eclipse

## Objective
To create and run a Maven project using Eclipse IDE and understand the Maven project structure.

## Steps Performed

1. Created a Maven project in Eclipse using the Quickstart archetype.
2. Configured the project with:
   - Group ID: com.fsad
   - Artifact ID: maven-eclipse-demo
3. Understood the purpose of key Maven components:
   - src/main/java: Contains application source code
   - src/test/java: Contains test cases
   - pom.xml: Manages project configuration and dependencies
4. Wrote a Java program to print:
   "Welcome to Maven using Eclipse IDE"
5. Executed Maven goals to run the project successfully.
6. Added a new Java class and re-ran the application to verify changes.

## Result
The Maven project was created and executed successfully in Eclipse.
