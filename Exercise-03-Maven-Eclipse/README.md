# Exercise 3: Creating and Running a Maven Project in Eclipse

## Objective
To create and run a Maven project in Eclipse IDE using the Quickstart archetype and understand the Maven project structure.

## Project Description
A Maven project was created using Eclipse IDE. A Java program was written to print a welcome message using Maven execution.

## Steps Performed

### 1. Maven Project Creation
- Created a Maven project using the maven-archetype-quickstart.
- Configured Group ID and Artifact ID.

### 2. Project Structure Explanation
- `src/main/java` : Contains the main Java application source code.
- `src/test/java` : Contains unit test source code.
- `pom.xml` : Maven configuration file containing project metadata and dependencies.

### 3. Maven Execution
- Executed the project using Maven Run As.
- Verified the output in the console.

### 4. Project Modification
- Added a new class and re-ran the application successfully.
