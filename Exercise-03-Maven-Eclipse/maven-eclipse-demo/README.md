# Exercise 3: Creating and Running a Maven Project in Eclipse

## Objective
To create and execute a Maven project using Eclipse IDE and understand the Maven project structure.

## Steps Performed

### 1. Maven Project Creation
- Created a Maven project in Eclipse IDE using the **Quickstart archetype**.
- Configured the project with:
  - **Group ID**: com.fsad
  - **Artifact ID**: maven-eclipse-demo

### 2. Understanding Maven Project Structure
- **src/main/java**  
  Contains the main application source code. The Java class that prints  
  **"Welcome to Maven using Eclipse IDE"** is placed here.

- **src/test/java**  
  Used for writing and maintaining unit test cases for the application.

- **pom.xml**  
  The Project Object Model file that manages project configuration, build
  details, and dependencies required by the application.

### 3. Running the Project
- Executed Maven goals using Eclipse and command line.
- Verified successful execution of the project and output message:
  **"Welcome to Maven using Eclipse IDE"**.

### 4. Project Modification
- Added a new Java class to the project.
- Re-ran the application using Maven goals to verify the modification.

## Result
The Maven project was created, executed, and modified successfully using
Eclipse IDE.
